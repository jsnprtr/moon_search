Apollo 11 Transcripts Search Engine

What is it?

A (work in progress) search engine that contains the Apollo 11 Technical Air-to-Ground Transcript. You can search for any term and any recorded transmission that matches the search term will be displayed in the "Results" panel. You can select a result by clicking on it, and it will appear in context in the "Results in Context" panel. Here you can scroll up and down to see more of the conversation.

Refreshing the page will load a random preset search term that might be interesting.

Why?

The transcripts exist online, and can be downloaded in PDF here from Nasa's History Division. I wanted to create a more interactive experience with the transcripts.

Technical Details

The search engine itself uses Solr as the backend. The site makes direct calls to the Solr instance using JQuery. Most of the website is plain JavaScript and HTML. The CSS framework is Fomantic UI (previously Semantic UI). The Solr instance is hosted on an Amazon AWS EC2 Micro instance (as is this website). This site and the search engine can be found on github.
